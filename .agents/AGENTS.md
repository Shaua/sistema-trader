# Regras do Projeto

- **Git Push Automático**: Sempre que fizer correções, atualizações ou modificações no código, você deve automaticamente adicionar os arquivos, criar um commit com uma mensagem descritiva (ex: `git add .`, `git commit -m "mensagem"`) e enviá-los para o repositório remoto (`git push`) utilizando o terminal, sem precisar perguntar ao usuário por permissão, a menos que ele solicite o contrário.
